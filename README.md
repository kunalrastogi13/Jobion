# Jobion 

Jobion ek web-based application hai jo modern technologies ke saath develop ki ja rahi hai.  
Is project ka purpose clean architecture, scalable codebase aur smooth CI/CD workflow ke saath development ko demonstrate karna hai.

---

## Tech Stack

- Frontend: React + Tailwind CSS  
- Backend: Node.js (planned / in progress)  
- Version Control: Git & GitHub  
- CI/CD: GitHub Actions
